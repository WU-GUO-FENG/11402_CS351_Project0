# TWO SUM 問題說明

## 問題描述
給定一個整數數組 `nums` 和一個整數目標 `target`，請你在數組中找出兩個數，使它們的和等於目標值 `target`。

你需要返回這兩個數的索引，並且假設每個輸入只會有一個解，且同樣的元素不能使用兩次。

## 例子
- 輸入: `nums = [2, 7, 11, 15]`, `target = 9`
- 輸出: `[0, 1]`
- 解釋: `nums[0] + nums[1] = 2 + 7 = 9`

## 解題思路
1. 使用哈希表來存儲數字及其索引。
2. 遍歷數組，對於每個數字，計算出需要的另一個數字，並檢查該數字是否在哈希表中。
3. 如果存在，則返回當前數字的索引和哈希表中對應數字的索引。

## 時間複雜度
此算法的時間複雜度為 O(n)，因為我們只需遍歷數組一次。
